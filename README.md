# Multi-layered Legislative Knowledge Management with Property Graphs

This repository includes the PG-Schema of legislative property graphs and the Cypher queries over the Italian Legislative Property Graph that we presented in the paper Multi-layered Legislative Knowledge Management with Property Graphs submitted to Transactions on Graph Data and Knowledge - DROPS.
